Matrix Toolkit Build 051 - Technician UI Refinement

- Main dashboard now focuses on Connectivity, WiFi, Reporting and Fleet.
- Router and LTE tools moved into a collapsible More Tools section.

- LTE page adds modem status, band and Cell ID when supplied by the cache.
