BUILD 036 - WEB SERVER CONTROL

Root cron:
- Installs matrix_update_worker.sh only in /etc/crontabs/root.
- Removes Matrix update worker entries from the uhttpd crontab.
- Prevents update and recovery jobs from running as the web-server user.
- Removes duplicate worker entries before installing one root entry.

uhttpd:
- All Matrix scripts use one central matrix-web-reload command.
- No Matrix CGI, Doctor, self-heal or ZeroTier worker directly restarts uhttpd.
- Uses an exclusive lock.
- Does nothing when the requested web configuration is already active.
- Reloads once when necessary.
- If reload fails, stops once, waits until ports are free and starts once.
- Prevents overlapping starts and repeated bind(): Address in use errors.

- Warns when an update worker remains in the uhttpd crontab.
- Checks that no packaged Matrix runtime script directly restarts uhttpd.
