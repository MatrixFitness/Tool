BUILD 030 - MATRIX FLEET DASHBOARD

Fleet Dashboard:
- Displays JNLMatrix, JBEMatrix and JDKMatrix in three responsive cards.
- Shows saved ZeroTier IP, live online status, Toolkit build, Node ID and last seen.
- Toolkit and Admin buttons automatically use the saved current IP.
- Adds Copy SSH command button.
- Highlights the router currently being viewed.

Fleet Discovery:
- Adds Scan Fleet button.
- Scans the 10.74.91.0/24 ZeroTier subnet in parallel.
- Identifies routers by the hostname returned by the Matrix status endpoint.
- Automatically updates fleet.conf when a router receives a new ZeroTier IP.
- Preserves discovered IP addresses across normal Matrix updates.
- Does not require the ZeroTier Central API.

Limitations:
- A newly reauthorized router must be online on the same ZeroTier network.
- The scan can only discover routers that have Build 030 status endpoint installed.
- The first scan after all routers are upgraded may use the supplied current IP seeds.
